# Theme

You can customize the theme of the widget. You need to create a project first.

## Theme button

Click the theme button on the project card, you'll be navigated to a theme editor.

![Dashboard of projects](/images/docs/features/theme/dashboard.png)

## Color options

Click the primary color button on the left, preview your change instantly on the right.

![Theme color](/images/docs/features/theme/theme-color.png)
