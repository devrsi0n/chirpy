# Get started

Chirpy comment widget can be integrated in your website easily.

## Usage on any website

To embed the comment widget on your website, first add the script tag with your project id to the HTML:

```html
<script
  defer
  src="https://chirpy.dev/bootstrap/comment.js"
  data-chirpy-pid="<your-project-id>"
>
<script>
```

Then, add the data-chirpy-comment attribute to any HTML element that should render the widget:

```html
<div data-chirpy-comment>...</div>
```
