# Analytics

You can view the stats of your website pages within Chirpy.

You need to create a project first to view the stats. Checkout [Get started](/docs/get-started) to create one.

## Basic stats

You can see the basic stats of your website on project card of dashboard, it shows the page views of yesterday.

![project card](/images/docs/features/analytics/project-card.png)

You can click it to view the details.

![analytics page](/images/docs/features/analytics/analytics-page.png)
