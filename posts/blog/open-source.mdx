Chirpy is a privacy friendly and customizable [Disqus](https://disqus.com) alternate. With Chirpy you can access your audience quickly and build a better community.

## Solution

To avoid privacy issues like Disqus has, we chose our tech stacks carefully, and we prefer to use self-hosted versions of other services that we don't implement them ourself. e.g., for analytics, we self-hosted a [plausible](https://plausible.io/) instance to avoid share your and your customer's data with third party companies. Plausible is a [open-source](https://github.com/plausible/analytics/) and [privacy friendly](https://plausible.io/privacy-focused-web-analytics) software as well, so you don't need a cookie consent banner because of using our services. We also plan to provide a self-host solution if you have privacy concern or just want to keep the data in your server.

For performance issues, we built our services with modern frontend technologies, e.g. [Next.js](https://nextjs.org/), we generated comment pages during the build process, thus your user will see the UI immediatelly instead of loading a bunch of JS & CSS and API calls to render the UI, it can be very solw on mobile devices.

## Currently state

Currently, we're in the public beta phase, it means basic features should be work, like social sign-in, create a project, integrate it in your website, comment and reply with rich-text-formating & markdown shortcuts, like a comment, etc. Please note: you may lose your data if there're some hard conflicts we can't migrate our database easily, but that should be rare.
